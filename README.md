# latex_scheleton

En este repositorio se presentan varias plantillas para entregas de documentos en \LaTeX .

## Plantillas Originales descargadas desde Overleaf

Plantillas para journal en IEEE y APA7, estas son las originales con algunos cambios agregados. Se pueden copiar y comenzar a editar para utilizarlas en los cursos.

* Plantilla_APA7
* Plantilla_IEEE

## Plantilla automatizada desde config (informes)

Plantillas para entregar laboratorios de elctrónica y tareas de las fases de los cursos utilizando los mismos documentos del contenido. Maneja algunos comandos para cambiar fácilmente entre los formatos editando el archivo config.tex dentro del directorio setup.

* APA7
* IEEE

## Plantilla para los informes de Laboratorio (Física)

Con el formato que piden en los cursos más la portada.

## Plantillas de presentaciones (Bemaer)

Los directorios que inicial con presentacion. Una presentación con los comandos básicos de \LaTeX se muestra en la presentacion_latex
